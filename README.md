# @primitivekit/react

Primitive React components with zero hardcoded values. Fully customizable via CSS variables.

## Installation

```bash
npm install @unstring/primitivekit-react
```

## Usage

```jsx
import { Button } from '@unstring/primitivekit-react';

function App() {
  return <Button variant="primary">Click me</Button>;
}
```

## Customization

```css
:root {
  --btn-primary-bg: #8b5cf6;
  --btn-border-radius: 2rem;
}
```

## Components

- **Button** - 6 variants, 3 sizes, loading states, icons, full accessibility
