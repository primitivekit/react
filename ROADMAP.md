## Development Principles

1. **Zero Hardcoded Values** - Everything via CSS variables (50-150+ per component)
2. **RGBA Colors** - All colors use `rgba()` format for transparency control
3. **TypeScript** - Complete type safety with comprehensive interfaces
4. **Accessibility** - WCAG 2.1 AA compliant (ARIA, keyboard nav, screen readers)
5. **Responsive** - Mobile-first design
6. **Customizable** - Every visual property controllable
